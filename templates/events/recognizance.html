RECOGNIZANCE

PS 1: 17th March 10AM - 19th March 11:59PM

PS 2: 19th March 10AM - 20th March 11:59PM

Team size: 1 - 2

Description

With an abundance of data and data-driven companies like Google and Facebook continuing to analyze and research intelligent machine algorithms, The domain of Machine learning has boomed like never before. Recognizance is a machine learning hackathon that thrives to give a real-world experience with machine learning and its implementations. This provides an opportunity to learn about the fundamental algorithms that are "state-of-the-art" solutions. The algorithms would mainly comprise of Supervised regression and classification. We mainly focus on simple Neural Networks, Computer Vision and NLP. To top it off it would be competition to squeeze out the least error and highest accuracy.

And may the Gradients be with you.

Workshops

  1. Basics of Python

    A brief introduction to Anaconda, Spyder and Jupyter. Introduction to Python (print, variables, Interpreter), Control Flow Tools (if-else, while, for, range, break), Function, Installing and importing modules and packages, File reading and writing, Basic Data Structures (Numpy array, Lists, Tuples, Dictionaries), Pandas, Matplotlib

  2. Concepts of Data Preprocessing

    A brief conceptual walkthrough will be given on how to examine patterns and preprocess large, noisy raw data into something more meaningful set of features that we would be fed into any machine learning algorithm. We would be briefing on scikit’s sklearn library and its functions like Normalize, Polynomial Features, Transforms, and using Matplotlib for Plots and histograms. We would also be touching on how to parse outliers in the dataset using Pandas.

  3. Machine Learning Algorithms

    We would be Introducing you to machine learning algorithms like Linear Regression, Logistic Regression, SVM, Decision Trees, Random Forests and be introducing to the core concepts of Loss, Entropy and will be demonstrating their working with the help of sklearn libraries.

  4. Introduction to Neural Networks

    With the advent of these powerful algorithms which are easily able to learn any linear or non-linear correlation between the features and labels we preferably choose this as our weapon of choice for large and complex datasets. We would be touching on the core mathematical intuitions of the neural networks, and would demonstrate on implementing a simple neural network architecture and train it using tensorflow.

  5. Dwelling into Deep Learning

    The extension of the simple neural network on Quadrillions of data points and using larger yet efficient architectures to learn these models gave rise to Deep Learning. We would be demonstrating how we can use Deep Learning Models in Tensorflow to Classify images and will give away the intuitional idea about CNNs ( Convolutional Neural Network), LSTMs ( Long Short Term Memory) and many more. We would also be briefing the main Problem Statement for Recognizance’21 in this workshop
